## 🎯 What Is This?

**The Nordic Media Observatory** tracks alternative news media across Denmark, Norway, Sweden, and Finland.

We've built:
- 📊 **Largest dataset** of Nordic alternative media content
- 🤖 **AI-powered analysis** using state-of-the-art NLP
- 🌍 **22+ years** of cross-national coverage
- 🔍 **Deep insights** into narrative patterns

## 🤖 The AI Behind It

### Model: **Mistral-Small-3.1-24B**
- 24 billion parameters
- State-of-the-art accuracy (MMLU 80.6%)
- 128k token context window
- Apache 2.0 license (fully open!)

### What It Does:
1. **Sentiment Analysis**: Positive/Neutral/Negative
2. **Topic Classification**: 11 news categories
3. **Entity Extraction**: Persons, organizations, locations

### Speed:
- **10-11 articles per second**
- vLLM inference (10-15x faster than standard)
- **100% coverage** (every article analyzed)

# Appendix: Technical Details
(For the nerds in the audience 🤓)

---

## 🔧 Technical Stack

### Data Collection:
- Web scraping from 68 outlets
- Custom extractors per outlet
- Deduplication and cleaning

### NLP Processing:
- **Model**: Mistral-Small-3.1-24B-Instruct-2503
- **Framework**: vLLM (optimized inference)
- **Hardware**: NVIDIA H100 80GB GPU
- **Speed**: 10-11 articles/second

### Storage:
- **Database**: PostgreSQL with JSONB
- **Format**: Parquet for analytics
- **Size**: ~782 MB (compressed)

### Analysis:
- **Language**: Python (Polars, Pandas)
- **Dashboard**: Streamlit
- **Visualization**: Plotly, Altair

---

## 📊 Data Quality

### Coverage:
- **100% NLP-enriched** (all articles)
- **85.2% partisan-labeled** (643,516 articles)
- **99%+ deduplication** rate

### Validation:
- Spot-checks on 100-article samples
- Category accuracy: ~85-90% (human evaluation)
- Sentiment accuracy: ~80-85% (compared to manual coding)

### Limitations:
- Some outlets have gaps in historical data
- OCR errors in older scanned content
- Non-Nordic language articles not fully analyzed
